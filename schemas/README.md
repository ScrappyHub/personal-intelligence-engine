\# PIE Schemas



This directory is reserved for canonical machine-readable schemas used by PIE.



\## Current state



Tier-0 correctness is currently proven primarily by:



\- deterministic scripts

\- canonical packet structure

\- independent verification behavior

\- golden vectors

\- transcripts and freeze artifacts



\## Intended future contents



Examples of schemas expected here later:



\- run seal schema

\- packet manifest schema

\- signature envelope schema

\- verification result schema

\- receipt schema



\## Rule



Schemas added here must be authoritative, versioned, and independently verifiable.
